> 由于 Vue3 不再支持 IE11，故而 ly-ui 也不支持 IE11 及之前版本。

## 通过 npm 或者 yarn 安装

**我们推荐使用包管理器的方式安装**，它能更好地和 [vite](https://vitejs.dev), [webpack](https://webpack.js.org/)
打包工具配合使用。

```shell
$ npm install ly-ui --save
```

```shell
$ yarn add ly-ui
```
